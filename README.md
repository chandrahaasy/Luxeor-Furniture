# Luxeor Furniture

## Project Description

This is a responsive furniture e-commerce website created using HTML5 and Tailwind CSS. It features a modern design with different sections for furniture products and home decor.

## Technologies Used

- HTML5
- Tailwind CSS

## Features

- Responsive design
- Hero section
- Product showcase
- Shop by Room
- Best Sellers
- Customer Testimonials
- Newsletter
- Footer

## How to Run

1. Download or extract the project folder.
2. Open `index.html` in any web browser.
